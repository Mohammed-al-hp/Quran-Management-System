نظام إدارة مراكز التحفيظ (Quran Management System)
هذا المشروع عبارة عن نظام متكامل لإدارة مراكز تحفيظ القرآن الكريم، يتكون من جزئين أساسيين تم تنظيمهما داخل مجلد المشروع الرئيسي:

1. تطبيق الهاتف (Mobile Application)
التقنية المستخدمة: Flutter.

الوظيفة: مخصص للمحفظين لمتابعة الطلاب، الحلقات، وتسجيل الحضور والغياب.

المسار: Antigravity/my_first_app/flutter_application_1.

2. نظام الإدارة والـ API (Web System)
التقنية المستخدمة: ASP.NET Core.

الوظيفة: لوحة تحكم إدارية لإدارة البيانات الأساسية وتوفير الخدمات (APIs) لتطبيق الموبايل.

المسار: Final Year Project/QuranCentersSystem.
